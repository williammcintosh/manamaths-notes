Place Values and Decimals

T1_LO1 · lo-yr9-place-values-and-decimals

Complete
Common mistake

Saying the value is just the digit. In $862$, the digit is $6$, but its value is $60$. In $2.74$, the digit is $7$, but its value is $0.7$.

Try these

  1. What is the value of the $4$ in $347$?

  2. What is the value of the $5$ in $5.3$?

  3. What is the value of the $4$ in $3.48$?
